AGM -- Active Grammar-based Modeling

A graph-grammar is a powerful tool that can be used to describe how a robot can build and modify its world model. Such information can be used for several purposes:

  • Generating plans to perceive the world
  • Generating plans to modify the world
  • Verifying world model modifications

AGM is a tool used to develop graph-grammars and compile them to PDDL, which is the de facto standard for AI planning. This is a necessary step since, in order to achieve our goals, we will be using AI planners.

Status

Still in development.

Download

Use the GIT repository: git://github.com/ljmanso/AGM.git, e.g.:

git clone git://github.com/ljmanso/AGM.git

Usage

An example AGM file can be found in the following link: grammarGualzru.pddl

Visual editor

python AGMEditor.py

Compiler

python agm2pddl <inputFile.agm> <outputFile.pddl>
